Prison Break

Status:
Unreleased
Genres:
Puzzle, stealth, single-player
Platform:
PC
Role(s):
Coding and Design

Prison Break is a 2D puzzle game with steal elememt, whose concept is based upon a famous American series with the same name. Player, as a prisoner, has to find their way to get out of pison by solving puzzles and acquiring keys, while at the same time avoiding security surveillance.

The game was completed within a semester of work, including both researching and coding phases, all by myself. Because of that, in order to cope with the limitation in members and time, only core functionalities were implemented. Other advanced ideas were saved for potential improvement patches in the future.

My work in this project consists of the following tasks:

  • Brainstorm ideas for game theme and art styles
  • Design game elements, such as: UI, map layout, challenges and enemies, items, and puzzles
  • Implement coding using LibGDX game framework (in Java language)
