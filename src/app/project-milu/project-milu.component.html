M.I.L.U is a multiplayer life-simulation game that had been up and running since 2008. It focuses on re-creating a peaceful and friendly community for players to relax, have fun, and bonding with like-minded people. Featuring in-game activies include: fishing, crafting, raising pets, trading, and more. There are no extra-hard difficulties, no secrets, no levels, and no potential harmful challenges to players.

I joined the team back in May 2019 for the server programmer position. My main responsibility revolves around handling server-side features as well as its connection to MySQL Database System. Nonetheless, from time to time, I also participate in developing client-side functions especially for major updates.

My contributions:

  • For server:
    • Implement and/or optimize server-side logic.
    • Monitor server status to react in time for unexpected issues.
    • Design database table and related stored procedure.
  • For client:
    • Implement functions to process data between client and server.
    • Make in-game UI assets for new features.
    • Update in-game map for weekly event.
