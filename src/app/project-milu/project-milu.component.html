M.I.L.U is a multiplayer life-simulation game that had been up and running since 2008. It focuses on re-creating a peaceful and friendly community for players to relax, have fun, and bonding with like-minded people. Featuring in-game activies include: fishing, crafting, raising pets, trading, and more. There are no extra-hard difficulties, no secrets, no levels, and no potential harmful challenges to players.

I joined the team back in May 2019 as server programmer, and had been working on this project for almost 3 years. My main responsibility lies in the implementation and optimization of server logic, as well as data manipulation via SQL database system. From time to time, however, I also participate in developing client-side behaviours especially for major upgrades.

Here are my contributions:

  • For server:
    • Implement and modify server-side logic using C/C++ language
    • Monitor status of server application in real time and react accordingly
    • Design table structure and create related functions/procedures in MySQL
  • For client:
    • Develop or improve features using in-house game engine (in C/C++ language)
    • Make game UI according to handed out design using in-house tool
