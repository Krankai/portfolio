Memor'ile

Prototype (2022)
Platform:
WebGL, Windows
Genres:
Casual, Puzzle
Link:
krankai.itch.io/memorile

Memor'ile is a simple puzzle game aims to give players a challenge for their memorizing abilities. Multiple special tiles will be briefly shown at the start then quickly hidden, which players have to remember and flip them up. The faster they can get it all, the higher score they can accumulate.

This game was made entirely through my solo effort using Unity Game Engine within the span of two (2) months. The workload includes:

  • Brainstorm game idea and concept.
  • Incorporate object-oriented programming design into the game structure.
  • Implement the game from scratch.
  • Search and utilize assets from vairous free open sources.
