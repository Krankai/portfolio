Dino Age

Status:
Unreleased
Genres:
Platformer, Single-player
Platform:
PC and Mobile
Role(s):
Coding and Design

Dino Age is a 2D singple-player platformer game setting in a mythical fantasy world. The main theme is about prehistoric age with dinosaurs and strange creatures. Players have to go through multiple stages and defeat different enemies in order to get the final reward.

This game was collaboratively developed by a 2-member team back in 2017 as an internship product. Due to the time constraint, we aimed to only implement a lightweight version with basic playable characters and few progressive stages. Nevertheless, there are still many key interesting features being added, for example, diverse enemies patterns and hidden puzzles.

All tasks were distributed equally and distinctly between members to avoid overlapping responsibilities. My contributions include:

  • Design in-game UI: main menu, character selection screen, skill upgrade screen, etc.
  • Design map layouts using Tiled Map Editor
  • Design item system
  • Design challenegs and enemies, including their behavior patterns
  • Implement relevant coding work using Cocos2d framework
